# AppleLog SPA 화면 구성 가이드 (UI Architecture)

현재 `index.html` 파일 하나에 모바일 기반의 SPA(Single Page Application) 형태로 통합된 **AppleLog (과수 농장 관리 앱)**의 전체 화면 구성을 안내합니다. 각 화면은 자바스크립트의 `switchView()` 함수를 통해 전환됩니다.

## 📱 앱 구조 및 레이아웃 요약

- **헤더 (Header)**: 상단에 위치하며 현재 화면의 타이틀을 표시하고 알림 버튼(종 모양)을 제공합니다. 스캔 화면에서는 다크 모드로 전환됩니다.
- **메인 컨텐츠 (Main View)**: 중간 영역으로, 아래 설명된 6개의 화면 중 하나가 표시됩니다.
- **하단 네비게이션바 (Bottom Navigation)**: 대시보드, 일지, 스캔 탭으로 구성되어 있으며 화면 간 빠른 이동을 돕습니다. (특정 화면에서는 자동으로 숨겨짐)

---

## 🎨 주요 화면(Views) 상세 가이드

### 1. 대시보드 화면 (`id="view-dashboard"`)
앱 실행 시 가장 먼저 표시되는 홈 화면입니다.
- **오늘의 상태 요약**: 현재 날씨 정보(온도, 상태)와 과수원의 종합적인 건강 상태를 시각적으로 보여줍니다.
- **할 일 목록 (Tasks)**: 오늘 해야 할 주요 농작업(예: 방제, 비료 주기 등) 리스트를 제공합니다.
- **긴급 알림 (Alerts)**: 병해충 발생 등 즉각적인 조치가 필요한 알림을 상단에 띄워줍니다.

### 2. 전체 일지 화면 (`id="view-logs"`)
지금까지 기록한 농작업 내역을 시간순으로 조회하는 화면입니다.
- **검색 바**: 특정 작업이나 날짜의 일지를 검색할 수 있습니다.
- **[새 일지] 버튼**: 우측 상단 플로팅 스타일 버튼으로, 새로운 일지 작성 화면으로 이동합니다.
- **일지 리스트**: 월별로 그룹화되어 있으며, 각 작업 카드에는 날짜, 카테고리 태그, 상세 내용, 첨부된 사진이 포함됩니다. 

### 3. 새 일지 작성 화면 (`id="view-new-log"`)
농작업을 수행한 후 세부 내역을 기록하는 폼(Form) 화면입니다.
- **입력 필드**:
  - **작업 카테고리**: 수확, 가지치기, 비료 등 셀렉트 박스 제공.
  - **날짜**: 작성 당일 날짜가 기본으로 기입됨.
  - **상세 내용**: 텍스트 입력 영역.
- **사진 추가 (업로드)**: 네이티브 `<input type="file">`을 활용하여 모바일 기기의 **카메라로 즉시 촬영**하거나 **갤러리에서 사진을 불러와** 실시간 미리보기를 제공합니다.
- **[일지 저장하기] 버튼**: 작성한 내용을 임시 저장하고, 전체 일지 목록 최상단에 카드를 즉시 추가합니다.

### 4. 성공 알림 화면 (`id="view-success"`)
일지 저장이 성공적으로 완료되었음을 알리는 피드백 화면입니다.
- **애니메이션 효과**: 시각적으로 눈에 띄는 체크 마크 아이콘이 팝업됩니다.
- **요약 정보 표시**: 방금 기록한 날짜와 작업 카테고리를 요약해서 보여줍니다.
- **[일지 목록으로 돌아가기] 버튼**: 클릭 시 다시 전체 일지 화면으로 되돌아갑니다.

### 5. AI 진단 스캔 화면 (`id="view-scan"`)
과수원의 병해충을 진단하기 위해 카메라를 실행하는 (시뮬레이션) 화면입니다.
- **카메라 뷰 파인더**: 화면 전체를 채우는 뷰 파인더 형태를 가지며, 몰입감을 위해 헤더가 검정색(다크 모드)으로 변합니다.
- **스캔 가이드라인**: 사과나 나뭇잎을 맞추기 위한 사각형 프레임과 타겟팅 애니메이션 제공.
- **촬영 버튼**: 하단의 큰 원형 버튼을 누르면 '분석 중' 로딩 후 진단 결과 화면으로 넘어갑니다.

### 6. 진단 결과 화면 (`id="view-diagnosis"`)
스캔 화면에서 촬영한 사진을 AI가 분석한 결과를 보여줍니다.
- **분석 요약 및 신뢰도**: 탐지된 질병(예: 화상병)의 이름, 심각도, 신뢰도 퍼센트를 붉은색 경고 박스로 강조합니다.
- **빠른 요약**: 해당 질병의 원인과 특성을 간략히 설명합니다.
- **치료 솔루션 (가이드)**: 긴급 조치 사항과 화학적 방제 방법 등을 단계별로 제공합니다.

## ⚙️ 자바스크립트 동작 로직 (`script` 태그 내)
- **`switchView(viewId)`**: 전달받은 `viewId`에 해당하는 `section`만 화면에 표시하고 나머지는 숨김 처리(`hidden` 클래스 토글)합니다. 화면 성격에 맞춰 하단 바를 숨기거나 헤더 스타일을 변경하는 로직이 포함되어 있습니다.
- **`previewImage(event)` & `clearPreview()`**: 새 일지 작성 시 사진을 업로드하면 `FileReader` 객체를 사용해 이미지를 즉시 미리보기 화면에 띄우거나, 취소 시 삭제합니다.
- **`saveLog()`**: 새 일지 화면에서 입력한 텍스트와 사진 데이터를 읽어 HTML 구조(`logHTML`)로 동적 생성한 뒤, 전체 일지 화면의 `april-logs-container` 최상단에 `insertAdjacentHTML`로 삽입합니다.
